# Auth Service

> Authentication and authorization service

## Overview

Handles user authentication, JWT token generation, and authorization.

## Tech Stack

- **Runtime:** Node.js 18.x
- **Framework:** Express 4.x
- **Database:** PostgreSQL 15
- **Cache:** Redis 7

## Quick Start

```bash
npm install
npm run dev
```
