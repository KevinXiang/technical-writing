# Shared Library

> Shared utilities and types for the E-Commerce Platform

## Overview

Common utilities, types, and helpers used across multiple services.

## Contents

- **Types** - TypeScript type definitions
- **Utils** - Common utility functions
- **Middleware** - Express middleware
- **Validators** - Input validation schemas

## Installation

```bash
npm install @ecommerce-platform/shared
```

## Usage

```typescript
import { User, validateEmail } from '@ecommerce-platform/shared';
```
